# 快速入门指南

## 5 分钟快速上手

### 1. 安装依赖

```bash
pip install yfinance pandas numpy requests finnhub-python
```

### 2. 运行第一个分析

```bash
python main.py AAPL
```

就这么简单！程序会自动分析苹果公司的股票并输出报告。

### 3. 配置新闻数据（可选但推荐）

为了获得更准确的分析结果，建议配置新闻 API：

1. 访问 https://finnhub.io/register 注册（免费）
2. 获取 API Key
3. 编辑 `config.py`，设置：
   ```python
   FINNHUB_API_KEY = "你的API密钥"
   ```

### 4. 分析更多股票

```bash
# 分析特斯拉
python main.py TSLA

# 同时分析多只股票
python main.py AAPL MSFT GOOGL NVDA

# 使用高级情绪分析（需要先安装 transformers 和 torch）
python main.py AAPL --finbert
```

## 理解输出结果

### 信号类型

- **🟢🟢 强烈买入**: 综合得分 > 60，建议积极建仓
- **🟢 买入**: 综合得分 30-60，可考虑建仓
- **🟡 持有**: 综合得分 -30 到 30，维持现有仓位
- **🔴 卖出**: 综合得分 -60 到 -30，建议减仓
- **🔴🔴 强烈卖出**: 综合得分 < -60，建议清仓

### 关键指标

- **RSI**: 
  - < 30: 超卖，可能反弹
  - 30-70: 正常区间
  - \> 70: 超买，可能回调

- **MACD**: 
  - 金叉: 看涨信号
  - 死叉: 看跌信号

- **均线排列**: 
  - 多头排列: 短期趋势向好
  - 空头排列: 短期趋势偏弱

- **成交量**: 
  - 放量 (> 1.5x): 市场活跃
  - 缩量 (< 0.5x): 观望情绪浓厚

## 实战建议

### 1. 分批建仓
即使是"强烈买入"信号，也建议分 2-3 次建仓，避免一次性买入。

### 2. 严格止损
按照报告中的止损位设置止损单，避免情绪化操作。

### 3. 关注大盘
个股分析需要结合大盘走势，如果大盘走弱，个股信号的可靠性会降低。

### 4. 定期复盘
每天或每周运行分析，跟踪持仓股票的信号变化。

### 5. 风险控制
- 单只股票仓位不超过 20-30%
- 总仓位根据市场环境调整
- 高风险股票（如 TSLA）降低仓位
